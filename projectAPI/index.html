AlterTail API v1.0 - Documentation

Read

Description
Returns all Users in JSON format.

HTTP request
GET http://localhost/AlterTail/projectAPI/user/read.php
In the case of local API installation, the exact URL may vary - please check your local setup.

Verification

Description
Checks whether the username entered exist and whether the password is correct.

HTTP request
GET http://localhost/AlterTail/projectAPI/user/verification.php?un=jimmyboi&pw=jimmy
where un is the username parameter, and
jimmyboi is the username value.
Additionally, pw is the password parameter, and
jimmy is the password value In the case of local API installation, the exact URL may vary - please check your local setup.
